Tasks provided by YCSAS

Task 1: Pushing Code to GitHub

In this task, I learned how to use Git and GitHub for version control and project management. I created a GitHub repository, initialized it with Git, and successfully pushed my project code. This task helped me understand repository creation, commit practices, tracking changes, and maintaining project versions. It also introduced me to professional development workflows and the importance of using version control in real-world projects.

Task 2: Simple Website Development

For this task, I designed and developed a simple website using HTML, CSS, and basic JavaScript. The website included structured content, basic styling, and navigation. Through this task, I learned how to organize web pages, apply styling for better visual appearance, and understand the basic flow of web development from structure to design.

Task 3: Registration Form

In this task, I created a functional registration form using HTML and CSS, with basic validation using JavaScript. The form included fields such as name, email, password, and contact details. I learned how to design user-friendly forms, validate user input, and ensure proper data collection. This task improved my skills in form handling, input validation, and improving user experience.

Task 4: To-Do List using DOM Manipulation

For this task, I developed a dynamic to-do list application using JavaScript and DOM manipulation. Users could add, edit, and delete tasks interactively. This project helped me understand how JavaScript interacts with HTML elements, how to handle events, and how to update the user interface dynamically without reloading the page. It significantly strengthened my logical thinking and problem-solving skills.

Task 5: Portfolio Website

In the final task, I designed and developed a complete personal portfolio website to showcase my skills, projects, and internship experience. The portfolio included sections such as About Me, Skills, Projects, Internship Details, and Contact Information. I focused on creating a responsive, visually appealing, and user-friendly design. This task allowed me to integrate all the technologies learned during the internship and present my work in a professional manner.
