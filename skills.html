MY SKILLS

HTML(HyperText Markup Language)

I learned how to create well-structured, semantic, and accessible web pages using HTML. This includes organizing content with appropriate tags, improving readability for both users and search engines, and ensuring proper document structure. I also worked with forms, tables, media elements, and learned best practices for writing clean and maintainable markup.

CSS(Cascading Style Sheets)

I developed strong skills in styling web pages using CSS to create visually appealing and responsive designs. This includes layout techniques such as Flexbox and Grid, responsive design using media queries, and styling components consistently. I also learned how to use animations, transitions, and hover effects to enhance user experience while maintaining a professional UI design.

JavaScript

I gained hands-on experience in adding interactivity and dynamic behavior to websites using JavaScript. This includes DOM manipulation, event handling, form validation, basic animations, and controlling page behavior based on user actions. I also learned to write clean and efficient scripts to improve website functionality and user engagement.

React

I learned the fundamentals of React, focusing on component-based architecture for building scalable and reusable user interfaces. This includes understanding JSX, props, state management, and basic hooks. React helped me improve my ability to structure applications efficiently and develop dynamic, fast, and interactive web applications.

Git & GitHub

I learned how to use Git for version control, enabling me to track changes, manage code versions, and collaborate effectively. Using GitHub, I gained experience in creating repositories, pushing and pulling code, maintaining project history, and following professional development workflows. This helped me understand collaboration and code management in real-world projects.

Excalidraw(UI Planning & Wireframing)

I used Excalidraw to create wireframes and design layouts before development. This helped me plan user interfaces effectively, visualize ideas, and structure web pages logically. UI planning improved development efficiency and ensured better user experience in final projects.
